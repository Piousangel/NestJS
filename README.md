Nest JS란?
- 효율적이고 확장 가능한 Node.js 서버 측 애플리케이션을 구축하기위한
  프로엠 워크. 프로그레시브 JavaScript를 사용하고 TypeScript로 빌드되고
  완벽하게 지원하며 OOP, FP, FRP 요소를 사용할 수 있게 해줍니다.

Nest JS의 내부
- 내부적으로 Next는 Express와 같은 강력한 HTTP 서버 프레임 워크를 사용하며
  선택적으로 Fastify를 구성할 수 있습니다
